# Knoppen

2 knoppen worden gebruikt om de richtingaanwijzers aan te sturen. Een links aan het stuur en een rechts zodat de fietser erop kan klikken als hij de richtingaanwijzers wilt aansteken. 

# Type knoppen

1. 2-pin knoppen, dit zijn mechanische knoppen met 2 pins. Standaard staat die op HOOG, als je klikt komt het in verbinding met GND, waardoor die op LAAG staat.:
- als je niks doet -> circuit is open
- als je klikt -> twee pins zijn verbonden



Wiring van een 2 pin knop:
- Pin 1: GND
- Pin 2: Digitale pin (bv. D2)
